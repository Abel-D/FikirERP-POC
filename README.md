# FikirERP-POC
This is a node.js server and reactjs client web application utilizing MongoDB database intended to serve as a proof of concept for the real application.

### server
#### Dependencies
mongoose 7.4.0
express 4.18.2
mongodb 5.7.0
mongoose 7.4.0
nodemon 3.0.1

## Run application

node index.js

### client
#### Dependencies

axios 1.4.0 
bootstrap 5.3.0
primeflex 3.3.1 
primeicons 6.0.1 
primereact 9.6.0 
react 18.2.0 
react-dom 18.2.0 
react-router-dom 6.14.2 
react-scripts 5.0.1
web-vitals 2.1.4

## build application

production
npm run build

## Run application

npm start
